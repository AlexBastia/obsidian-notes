The **Unified Modelling Language** is a language used to standardize and aid the description of systems.

Classifiers:
- Class
- Interface
- Datatype
- Component
- Node
- Use Case
- Subsystem

Diagrams:
- Structural:
   - Class
   - Object
   - Component
   - Deployment
- Behavioural:
   - Use Case
   - Sequence
   - Collaboration
   - Statechart
   - Activity
